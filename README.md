# Nexus Digital - Landing Page Premium

Bienvenido al proyecto de landing page premium para **Nexus Digital**. Este sitio fue diseñado con una estética "Dark Luxury", optimizada para empresas de tecnología y desarrollo de software.

## Sistema de diseño
- **Estilo:** Dark Tech / Premium SaaS
- **Colores:**
  - Primario: Azul eléctrico (`#3B82F6`)
  - Secundario: Naranja vibrante (`#F97316`)
  - Fondo: Negro puro (`#000000`) y gris zen (`#09090B`)
- **Tipografía:**
  - Plus Jakarta Sans (títulos)
  - Inter (cuerpo)

## Estructura del proyecto
- `index.html`: estructura semántica y optimizada para SEO.
- `styles.css`: estilos modernos con glassmorphism y animaciones avanzadas.
- `script.js`: lógica interactiva (partículas, scroll reveal y contadores).
- `/images/`:
  - `logo.png`: identidad visual de la marca.
  - `hero_visual.png`: arte principal 3D.
  - `service_1.png` a `service_6.png`: imágenes temáticas para cada servicio.

## Características principales
1. **Hero section impactante:** visual 3D animado y efectos de glow.
2. **Tarjetas con efecto flip:** los servicios se presentan en tarjetas 3D que se revelan al pasar el cursor o al tocar en móviles.
3. **Scroll reveal:** las secciones aparecen suavemente a medida que el usuario navega.
4. **Sistema de partículas:** fondo dinámico con partículas de luz flotantes.
5. **Diseño responsive:** optimizado para desktop XL, tablets y dispositivos móviles.
6. **Contadores animados:** las estadísticas de la empresa se incrementan visualmente al llegar a la sección.

## Instrucciones de uso
1. Abre el archivo `index.html` en cualquier navegador moderno.
2. Para publicarlo, sube todos los archivos y la carpeta `/images/` a tu servidor o plataforma de hosting preferida, como Netlify, Vercel o GitHub Pages.
